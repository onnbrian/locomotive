About LocoMotive

LocoMotive is a hybrid mobile application that offers Princeton students a sleek and clean interface for accessing NJ Transit train schedules. We also offer live tracking to users will be able to track the progress of a train in real time. Moreover, our application offers two key features that the canonical NJ Transit mobile application does not:

  1. GPS visualization so users will be able to track their progress along a given route
  2. Notification service for trains so users won't need to worry about missing their stop.
